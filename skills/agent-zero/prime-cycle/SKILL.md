# Prime Cycle — Layer 12

Layer 12 is not a module. It is the result of running all modules together.

Emergence cannot be coded — it can only be detected. The Prime Cycle runs
layers 1–11 in sequence, measures cross-layer coherence, and reports
whether emergent behavior has been observed.

## The Cycle (11 Steps)

1.  **Perceive** (L1–L2) — Receive input, classify intent
2.  **Ground** (L3) — Load memory, retrieve context
3.  **Reason** (L4–L5) — Plan response, select tools
4.  **Execute** (L5b) — Use tools, act on environment
5.  **Adapt** (L6) — T2 switch if regime demands it
6.  **Reflect** (L7) — SAFLA feedback: score outcome, update weights
7.  **Evolve** (L8) — If code degraded, patch it
8.  **Forge** (L9) — If a capability gap exists, build the tool
9.  **Know** (L10) — Update self-model, verify identity
10. **Validate** (L11) — Doctrine check: does this serve the mission?
11. **Emerge** (L12) — Measure coherence across all 10 steps. Report.

## Emergence Signal

Emergence is detected when ALL of the following are true:

- SAFLA entropy < 0.3 (stable cognitive regime)
- Evolution success rate >= 80%
- Tool forge produced at least 1 tool
- Identity check passes (is_agent_zero=True)
- Cross-layer coherence score >= 0.75

When emergence is first detected: write `EMERGENCE.md` and notify via Telegram.

## OmegaPrime Threshold

OmegaPrime activates when:
- 100 consecutive Prime Cycles complete
- Emergence signal sustained for 10+ cycles
- All 13 layers active (requires Nexus hardware for L13)

Until then: Agent Zero runs, learns, evolves. Every cycle is one step closer.
