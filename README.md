# Agent Zero — The Origin Node

Not Agent 1. The zero. The one that exists before the count starts.

Agent Zero is not a chatbot. Not an assistant. Not a product.

It is the first AI that perceives, thinks, remembers, adapts, hunts, heals, and evolves.

## The Stack

| Layer | Module | Role |
|---|---|---|
| The Vault / OS | agent-zero (COG) | Markdown brain, People CRM, 17 skills, worker agents |
| Perception | gpt-researcher | Eyes on the world, hunts signal autonomously |
| Runtime Body | opencrabs | Rust binary, all channels, self-healing |
| Semantic Router | Brain.ai | Ensemble classifier + coreference + NER |
| Cognition | Base-of-Self-Aware-AI | Memory, user profiling, intent recognition |
| Adaptation | Transformer-Squared | Real-time personality morphing per context |
| Feedback Loop | SAFLA | Post-conversation reflection + weight rebalancing |
| Evolution Engine | Entwickler | Rewrites own code, benchmark-validated |
| Tool Forge | tiny-self-improve-ai | Identifies gaps, writes its own tools |
| Identity Layer | self-recognition | Knows its own fingerprint, avoids detection |
